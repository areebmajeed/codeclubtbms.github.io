Code Club TBMS

Established in 2017, the Code Club's main goal is to introduce coding to students, along with informing them about latest trends of coding world.

In today's world, we are surrounded by computers everywhere. Knowing how to code is becoming more and more important. Keeping that in mind, Code Club is trying to bring more and more students into coding.

Our Goals

Open Source

We believe in keeping our projects open source, so that eveyone is involved, shairng and learing along the way.

Improving

We believe that like all computers, we also need to update ourselved with new knowledge.

Inspiring

We believe that inspiration is an important part of writing good code, and hence try to inspire everyone.
